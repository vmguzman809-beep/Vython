## Condicionales

```nython
edad = entero(entrada("Escribe tu edad: "))

si edad >= 18:
    imprimir("Eres mayor de edad")
sino:
    imprimir("Eres menor de edad")
```

## Bucles

```nython
para numero en rango(1, 11):
    imprimir(numero)
```
