# Introducción

Ñython permite escribir programas con palabras clave en español y ejecutarlos sobre Python real. El objetivo es reducir la fricción inicial para estudiantes hispanohablantes sin aislarlos del ecosistema de Python.

La idea central es simple:

```text
Código Ñython -> Traducción -> Código Python -> Ejecución
```

Esto conserva compatibilidad con bibliotecas existentes y permite compilar a `.py` cuando se quiera ejecutar Python directamente.
