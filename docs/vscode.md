# Extensión de VS Code

La carpeta `vscode-extension/` contiene una base de extensión para trabajar con archivos `.ny` y `.nython`.

## Funciones actuales

- Reconoce archivos `.ny` y `.nython`.
- Aplica resaltado de sintaxis básico.
- Agrega snippets para `si`, `para`, `funcion` y `repetir`.
- Agrega comandos:
  - `Ñython: Ejecutar archivo actual`
  - `Ñython: Traducir archivo actual`
  - `Ñython: Compilar archivo actual`
